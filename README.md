# CR Advocates LLP – Virtual Law Firm Platform

A full-scale legal SaaS platform serving Africa, built with MERN stack.

## Tech Stack
- React + Tailwind
- Node.js + Express
- MongoDB
- Resend (Emails)
- Cloudinary (Files)
- Paystack / Flutterwave / Stripe
- Socket.io (Real-time)

## Features
- Legal Ecommerce
- Client Portal
- Retainer Subscriptions
- Contract Automation
- Booking & Virtual Meetings
- Admin Dashboard
- Multilingual + PWA

## Setup
1. Clone repo
2. Setup backend
3. Setup frontend
4. Configure `.env`
5. Run both servers
